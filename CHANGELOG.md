# Changelog

All notable changes to this project will be documented in this file.

## [0.9.3] - 2026-05-30
### Added
- Pull request sidebar unread tracking with opt-in PR list loading.
- Annotation proofread coding agent support.

### Changed
- Improved mobile reader navigation.
- Reader archive data is stored under the launch directory.
- App title now uses the book emoji.

### Fixed
- HTML sidebar links are marked as read when opened.
- Reader can start outside a Git checkout.

## [0.9.2] - 2026-05-25
### Added
- Archive file action.
- Sandboxed HTML preview links.
- Emoji SVG favicon.

### Changed
- Clarified header copy actions.
- Documented CLI options.
- Ignored local Codex config files.

### Fixed
- Missing write directories are created on startup.

## [0.9.1] - 2026-05-23
### Added
- Read/write pane visibility toggles.
- Unread file tracking.

### Changed
- HTML sidebar files open in new tabs.
- Removed worktree display UI.

## [0.9.0] - 2026-05-17
### Added
- Initial public release of the reader document viewer.
- Local Markdown and text browsing through a browser UI.
- `main` branch diff display.
- Multiple read/write roots, file filtering, and browser-based editing.
